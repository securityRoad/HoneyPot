### 功能概括
- 蜜罐功能
- 日志采集（不依赖WEB容器的日志采集）
- 第三方账号获取
- 捕获攻击行为
- 自定义攻击行为
- 自定义第三方账号获取
- 触发攻击行为URL跳转（payload无法执行）
- 暴力穷举（目录/文件/参数/管理登录）识别
